Matthew Maurer (Jul 07 2025 at 21:40):

If I have

struct Foo {
  bar: usize,
  baz: usize
}

and I want to present through debugfs bar, baz, and some summary info bang that's a combo of the three, with the File version, that's either going to be written

File<File<File<Foo>>>

or

struct Foo {
  bar: File<usize>
  baz: File<usize>
}
let bang: File<Foo>

Either of those representations is kind of wacky compared to the owned variant, which would look something like

struct Boom {
  foo: Arc<Foo>,
  #[cfg(DEBUG_FS)]
  _debug: (File, File, File)
}
