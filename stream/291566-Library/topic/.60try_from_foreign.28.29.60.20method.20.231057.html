Valentin Obst (Jan 23 2024 at 08:00):

Hi, a default implementation is an implementation of a trait method inside the trait definition. Any type that implements the trait and does not supply an implementation of a method with a default implementation will use the default implementation. Also have a look here: https://doc.rust-lang.org/stable/book/ch10-02-traits.html#default-implementations
