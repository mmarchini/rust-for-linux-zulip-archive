Alice Ryhl (May 14 2024 at 15:40):

Another thing you can do is something along these lines:

unsafe trait DevresCompatible {
    fn _devres_get_callback() -> extern "C" fn(*mut c_void);
}

macro_rules! impl_devres_compatible {
    ($t:ident) => {
        static CALLBACK: extern "C" fn(*mut c_void) = $crate::devres::devres_callback::<$t>;

        unsafe impl DevresCompatible for $t {
            #[inline(always)]
            fn _devres_get_callback() -> extern "C" fn(*mut c_void) {
                CALLBACK
            }
        }
    };
}

Alice Ryhl (May 14 2024 at 15:41):

Each type used with DevRes then invokes that macro. The static is guaranteed to not get monomorphized.
