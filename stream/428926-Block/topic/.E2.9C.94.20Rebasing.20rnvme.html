Andreas Hindborg (Mar 10 2026 at 08:30):

Hi @SeungJong Ha ! I am currently rebasing on v6.19. I am doing some changes to pci/irq to support reconfiguration of the interrupt vectors. Allocating the vectors up front is not going to work for many devices. We need to be able to be able to bring up the admin queue, execute controller identify, set admin queue limits, and then set the queue count. If you allocate vectors up front, you risk allocating more vectors than the device supports.

Andreas Hindborg (Mar 10 2026 at 08:31):

You will see issues on systems that has more cores than IO queues.
