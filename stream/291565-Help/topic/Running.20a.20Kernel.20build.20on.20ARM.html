Aswin Unnikrishnan (Jun 01 2024 at 16:16):

Setting the VM for Kernel development

1.     Ubuntu server loads into a console shell

2.     Install GUI for the ubuntu server using

3.     Sudo apt install slim

4.     sudo apt install ubuntu-desktop and then select slim

5.     Reboot the sever to get to the GUI

6.     Login to the server with the username and password set during the boot

Setting up linux and other dependencies

1.     Sudo apt install git make

2.     Git clone –depth=1 https://github.com/torvalds/linux.git

3.     Git clone https://github.com/mirror/busybox.git

4.     Install rust following : https://www.rust-lang.org/tools/install
a. curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
b. echo PATH=$PATH:~/.cargo/bin >> ~/.bashrc 
c. source ~/.bashrc 

5.     rustup override set $(scripts/min-tool-version.sh rustc) 

6.     rustup component add rust-src rustfmt clippy

7.     make rustavailable

8.     sudo apt install gcc clang flex bison

9.     cargo install --locked --version $(scripts/min-tool-version.sh bindgen) bindgen-cli

Building the kernel

  1. sudo apt install gcc-x86-64-linux-gnu g++-x86-64-linux-gnu

  2. sudo apt install libelf-dev

  3. make ARCH=x86_64 CROSS_COMPILE=x86_64-linux-gnu- -j12

Kernel: arch/x86/boot/bzImage is ready  (#1)

Building initramfs

1.     cd ../busybox

2.     make ARCH=x86_64 CROSS_COMPILE=x86_64-linux-gnu- -j12

3.     make ARCH=x86_64 CROSS_COMPILE=x86_64-linux-gnu- -j12 install

4.     Set CONFIG_TC=n in .config if make install throws an erorr

  1. Create the initramfs structure

a.     mkdir initramfs

b.     mkdir -p initramfs/bin initramfs/sbin initramfs/etc initramfs/proc  \ initramfs/sys initramfs/dev initramfs/usr/bin initramfs/usr/sbin

c.     cp -a  _install/* ./initramfs/

d.     cd initramfs/

e.     vim init

#!/bin/sh

mount -t devtmpfs devtmpfs /dev

mount -t proc none /proc

mount -t sysfs none /sys

cat <<!

Welcome to Micro Linux!

Boot took $(cut -d' ' -f1 /proc/uptime) seconds

!

exec /bin/sh

f.      chmod a+x init

6.     find . -print0 | cpio --null -ov --format=newc | gzip -9 > ../initramfs.cpio.gz

Running the new kernel in a VM

1.     cd ../../linux/

2.     qemu-system-x86_64 -machine q35 -kernel arch/x86/boot/bzImage -initrd ../busybox/initramfs.cpio.gz
