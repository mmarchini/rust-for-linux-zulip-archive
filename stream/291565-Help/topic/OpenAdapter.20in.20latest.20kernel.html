wctrl (Mar 15 2025 at 18:44):

Hi there,

There's an unmerged PR on github with debugfs that uses OpenAdapter. However rust/kernel/file.rs was removed somewhere in v6.1, instead rust/kernel/fs/file.rs was introduced without any OpenAdapter functionality. Are there any pending patches or reasons why it was removed?

Miguel Ojeda (Mar 15 2025 at 20:06):

Code gets upstreamed to the Linux kernel as needed, since we cannot push unused code. v6.1 was the time Rust got merged into the kernel, but not everything was submitted. So it is rather that it got never added, rather than that it was removed.

Please see e.g. https://rust-for-linux.com/branches#rust and https://rust-for-linux.com/contributing#submitting-new-abstractions-and-modules for more information.

I hope that clarifies.
