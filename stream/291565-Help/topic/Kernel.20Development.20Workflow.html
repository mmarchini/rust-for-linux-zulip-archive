Alice Ryhl (Aug 05 2024 at 19:53):

How I test stuff depends heavily on the details. Generally, the changes I make are for drivers that will run in the Android kernel, so if I'm uncertain about a patch, I'll cherry-pick it into my Android tree and see if it works when used together with the real driver that it is intended for. This may involve running it on an Android emulator or a real phone. For many patches, I'm happy if it passes make LLVM=1 CLIPPY=1 and make LLVM=1 rustdoc.
