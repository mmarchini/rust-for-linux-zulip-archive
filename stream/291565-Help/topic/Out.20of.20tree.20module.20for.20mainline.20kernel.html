Miguel Ojeda (Jun 20 2023 at 09:22):

The issue is that rustc creates temporary files in the current working directory by default (even if --emit is passed, i.e. it does not use those paths for the temporary files, so I think we need to control the temporary files via --out-dir).

Miguel Ojeda (Jun 20 2023 at 09:22):

(and the CWD is not the right place in the out-of-tree case)
