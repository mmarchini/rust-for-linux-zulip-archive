Andreas Hindborg (Jun 20 2023 at 07:33):

I am not familiar with that API, but if possible, I think it is better to access the fields with the C API. That way you know locking is correct. If you access from Rust you need to make sure that you are serializing access correctly. I don't think you can do that without some code. You also need to consider that C side of kernel can change over time, if you use C API to access, you are better protected from that.
