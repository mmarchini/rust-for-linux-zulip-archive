Topic: what is rust metadata?

Baul (Jianguo Bao) (Sep 03 2023 at 04:31):

The kernel tree (KDIR) requires the Rust metadata to be available. These are generated during the kernel build

bjorn3 (Sep 03 2023 at 16:09):

@baul When rustc compiles a crate, it will emit not just the compiled object file(s), but also a metadata file (either as part of the .rlib file or as separate .rmeta file) which describes the public interface of the crate as well as the definitions of all #[inline] and generic functions. This file is then used by later rustc invocations when depending on this crate. In a sense it is similar to header files in C, except that they are auto-generated by the compiler when compiling and are in a binary format.
