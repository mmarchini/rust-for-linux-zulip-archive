Martin Rodriguez Reboredo (Jan 11 2023 at 20:29):

Currently Bindgen isn't capable of generating bindings from inlined functions directly. So to make use of them these are added to a helpers file that contain wrappers for those inlined functions, all prefixed with rust_helper_ (see rust/helpers.c), and then that file gets bindings generated. For example, to wrap the to_usb_interface inline function the following lines should be added to rust/helpers.c

struct usb_interface *rust_helper_to_usb_interface(struct device *d) {
    return to_usb_interface(d);
}
EXPORT_SYMBOL_GPL(rust_helper_to_usb_interface);

Miguel Ojeda (Jan 11 2023 at 21:03):

rusttest runs in the host, so kernel APIs cannot be used there yet. Instead, you may use doctests, those are already plugged into KUnit and run as any other KUnit test. Another alternative is creating a small test module, of course. We have been putting them in samples/rust/.
