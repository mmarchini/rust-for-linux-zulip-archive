rustavailable only shows if the toolchain is available, not whether CONFIG_RUST is enabled or not (which, as @y86 points out, has some requirements), please see the help target and the Quick Start guide:

Rust targets:
  rustavailable   - Checks whether the Rust toolchain is
                    available and, if not, explains why.
