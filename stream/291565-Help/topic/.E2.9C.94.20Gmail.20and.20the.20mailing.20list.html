Miguel Ojeda (May 19 2022 at 19:08):

Gmail's plain text mode is fine for normal messages, but it does not work for patches themselves because Gmail will modify the contents (e.g. by wrapping the lines).

Miguel Ojeda (May 19 2022 at 19:09):

IIRC some years ago there was a way to avoid the wrapping even from the web interface (I think I sent a patch that way), but nowadays I am not aware of any way of avoiding it.

Miguel Ojeda (May 19 2022 at 19:12):

So I recommend you either use a client like Mutt which also easily allows you to import patches as emails from e.g. git format-patch, or using git send-email.

Miguel Ojeda (May 19 2022 at 19:13):

Having said that, for the moment, if it is for Rust for Linux, you can use GitHub's PRs until we get merged.
