Andreas Hindborg (Feb 29 2024 at 09:32):

@Ariel Miculas if you still want to expand something, what you can do is find the .cmd file for your rust object. It has the command line used to invoke rustc. You can then add -Zunpretty=expanded and RUSTC_BOOTSTRAP=1 to that command line and run it in a shell. That works.

Miguel Ojeda (Mar 04 2024 at 18:54):

Andreas Hindborg said:

So it is rustfmt that is crashing, not rustc

Minimized:

const _: () = builtin # offset_of(x, x);

rustfmt ICEs (apparently on purpose) on certain expanded -Zunpretty=expanded constructs (e.g. offset_of!).

Reported it here: https://github.com/rust-lang/rustfmt/issues/6105.
