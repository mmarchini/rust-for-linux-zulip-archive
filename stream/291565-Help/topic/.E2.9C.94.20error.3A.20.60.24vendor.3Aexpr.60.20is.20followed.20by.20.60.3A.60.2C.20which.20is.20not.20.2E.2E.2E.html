I am working on HID bindings and a driver as documented here: https://binary-eater.github.io/tags/usb-monitor-control/

I have this macro.

#[macro_export]
macro_rules! usb_device {
    (vendor: $($vendor:expr), product: $($product:expr)) => {
        DeviceId(Opaque::new(bindings::hid_device_id {
            bus: 0x3, /* BUS_USB */
            vendor: $($vendor),
            product: $($product),
            // SAFETY: The rest is zeroed out to initialize `struct hid_device_id`,

Note that in that chapter, there is a star (*) after the comma, that means that it can match the inner part of the parenthesis any number of times (including 0), while also matching a separating comma between matches.
