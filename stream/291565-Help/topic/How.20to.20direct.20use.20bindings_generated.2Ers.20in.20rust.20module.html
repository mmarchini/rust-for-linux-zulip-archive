Alice Ryhl (Jun 12 2024 at 10:47):

So, your extern "C" block is how you would do it if you were not using bindgen. What you need to do is to add the header file you shared in include/linux/mentor.h, and then you add that file to the list in rust/bindings/bindings_helper.h. Once you do that, crate::bindings::mentor_write will let you access the function.

Alice Ryhl (Jun 12 2024 at 10:48):

However you will still get the same error, because the error means that mentor_read is not implemented anywhere. You need an actual implementation somewhere for that error to go away.
