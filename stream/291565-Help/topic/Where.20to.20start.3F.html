Benno Lossin (May 21 2023 at 16:01):

See https://rust-for-linux.com/branches if everything that you need is already on mainline then use one of Linus' stable tags, otherwise use rust-next. If you still need some code that is not in rust-next, but in the rust branch, then there is still some work to be done, since many of the things left in that branch are not ready to be upstreamed. Feel free to tell us what you need and we will see how long it will take to upstream.
