Please note that you have errors about clang not being found. Can you build a non-Rust-enabled LLVM=1 kernel?

I just tried in a fresh Ubuntu 24.04, and I could build the kernel.

Please note that you need to setup first LLVM (including Clang) properly, and then, to enable Rust, you need to make sure you are using the versioned packages from Ubuntu (e.g. pass Make BINDGEN=bindgen-0.65 RUSTC=rustc-1.80 etc.) and finally check that your dependencies are met (e.g. with Ubuntu's Rust 1.80, you will need to take care of disabling CONFIG_CALL_PADDING if you were trying to use a defconfig, so it may be easier to just use rustup to get at least Rust 1.81).
