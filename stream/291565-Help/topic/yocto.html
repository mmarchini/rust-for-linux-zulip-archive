Andreas Helbech Kleist (May 10 2024 at 05:16):

In yocto, you need to add all tools needed during compilation to DEPENDS as "-native" dependencies. So in this case you need to add something like rustc-native to DEPENDS to the kernel recipe. I'm not sure what the exact name of the recipe providing rustc is. You can answer that question by asking oe-pkgdata-util about which package provides /usr/bin/rustc.

Andreas Helbech Kleist (May 11 2024 at 05:55):

You shouldn't need meta-rust or meta-rust-bin if you're using a recent version of yocto. I just checked. It seems like what you need is rust-nativein DEPENDS, to get access to rustc.
