Louis Chauvet (Mar 07 2023 at 08:21):

Hello,
To enable rust you must run make LLVM=1 menuconfig (or your prefered configuration tool), don't forget the LLVM=1
If you can't see it, just search for the symbol CONFIG_RUST, it will shows the dependencies needed for rust, maybe you enabled one of them

For example you can see here that by default MODVERSIONS is enabled and RUST can only be active if MODVERSION is disabled.

If you want to activate the minimum set of features to run the kernel in qemu and use Rust, you can run make LLVM=1 allnoconfig qemu-busybox-min.config rust.config(taken from https://www.jackos.io/rust-kernel/rust-for-linux.html#dependencies)
