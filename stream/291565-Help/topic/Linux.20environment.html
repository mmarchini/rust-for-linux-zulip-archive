Ariel Miculas (Apr 27 2023 at 10:07):

I managed to get it to work, there were a few issues:

  • I was missing CONFIG_RD_GZIP=y in my config (so I was actually able to use a cpio archive as initrd, but not a gzip compressed one)
  • I needed to create an init script in the image with the following contents:
#!/bin/sh
busybox mount -t devtmpfs none /dev
busybox setsid sh -c 'exec sh -l </dev/ttyS0 >/dev/ttyS0 2>&1'
  • I modified qemu to run -kernel ../linux/arch/x86/boot/bzImage instead of -kernel ../linux/vmlinux and to use the new init script rdinit=/init

I still can't use -kernel ../linux/vmlinux, for some reason qemu doesn't like it (it doesn't get past SeaBios and Booting from ROM)
