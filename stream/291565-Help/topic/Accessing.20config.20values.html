Topic: Accessing config values

Alice Ryhl (Aug 24 2022 at 09:03):

I'm trying to access the config value CONFIG_ANDROID_BINDER_DEVICES from Rust code. In the binder.c file, it's used like this:

char *binder_devices_param = CONFIG_ANDROID_BINDER_DEVICES;
module_param_named(devices, binder_devices_param, charp, 0444);

What is the equivalent Rust code?

Martin Rodriguez Reboredo (Aug 24 2022 at 12:35):

There's a file that contains the Kconfig values for Rust at include/generated/rust_cfg for them to be passed to rustc and accessed with #[cfg(...)], #[cfg_attr(...)] or cfg!, Rust as of now doesn't have a method to access cfg values in the std library, so you may have to implement a macro that looks for a config in the rust_cfg file.
