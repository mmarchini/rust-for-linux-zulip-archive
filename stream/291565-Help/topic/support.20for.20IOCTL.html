Louis Chauvet (Mar 16 2023 at 13:37):

You can found a usage of IOCTL in this sample: https://github.com/Rust-for-Linux/linux/blob/rust/samples/rust/rust_semaphore.rs

Some macro like _IOR / _IOW / _IOC are not present in rust, so you may need to write "magic constants" to handle commands.

Alice Ryhl (Mar 16 2023 at 13:40):

You can use a C header file to define the commands. Binder does that.
