bjorn3 (Jun 19 2023 at 23:44):

Qemu doesn't support icmp (ping) for the userspace network implementation. Tcp and udp should work fine. If you really need icmp to work you will have to setup a tun/tap device and tell qemu to use it which requires root and last time I tried didn't manage to get working.
