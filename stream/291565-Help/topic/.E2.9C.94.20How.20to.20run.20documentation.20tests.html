Yutaro Ohno (Sep 24 2022 at 15:36):

Hi. I'm trying to run documentation tests, but it looks like make rusttest doesn't run doc tests. How to run them?
Specifically, I want to run doc tests in rust/kernel/str.rs.

Miguel Ojeda (Sep 24 2022 at 16:37):

The doc tests are run via KUnit, so you will want to boot the kernel to run them (see e.g. https://docs.kernel.org/dev-tools/kunit/start.html#running-tests-without-the-kunit-wrapper) or use the KUnit Python script.

Make sure you enable RUST_KERNEL_KUNIT_TEST -- see its description for details.
