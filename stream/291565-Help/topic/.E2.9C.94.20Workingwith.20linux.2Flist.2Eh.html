Benno Lossin (Aug 22 2024 at 05:37):

In Rust, we don't use the C structs directly, since that would involve a lot of unsafe code. Instead we use abstractions that allow us to write safe code.

The linked list abstractions have been in review for the last months and will be getting in this cycle.
