Topic: `tc-build` now has Rust support

Miguel Ojeda (Mar 01 2026 at 10:55):

tc-build ("Toolchain build scripts") just merged the ability to build Rust toolchains:

https://github.com/ClangBuiltLinux/tc-build

So now you can just do e.g.

./build-llvm.py --ref llvmorg-21.1.8 && ./build-rust.py --ref 1.93.1

to easily build a LLVM+Rust toolchain suitable for kernel development.

Or, for the upcoming Rust 1.94.0 release which supports LLVM 22 released a few days ago, you may write:

./build-llvm.py --ref llvmorg-22.1.0 && ./build-rust.py --ref origin/beta

For context, tc-build scripts aim to simplify the process by handling configuration of the different build systems, by providing good defaults and by handling downloads. They also make builds faster and smaller by avoiding building pieces not needed for kernel development.

The scripts come with options to customize the process. For instance, one can build LLVM toolchains with LTO, BOLT and/or PGO optimized for kernel development in particular. tc-build is how the very fast kernel.org LLVM toolchains are built!

More options or features for the Rust part may be added later on, but for now the basic support is there. If something doesn't work, please let us know.

We hope that helps!
