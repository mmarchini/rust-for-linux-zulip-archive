Topic: kernel.org LLVM + Rust toolchains

Miguel Ojeda (May 17 2024 at 19:38):

kernel.org is now hosting prebuilt stable versions of LLVM+Rust toolchains, thanks to Nathan Chancellor (ClangBuiltLinux):

  • https://kernel.org/pub/tools/llvm/rust/

These are the same slim and fast LLVM toolchains provided there in the parent folder with versions of Rust added to them that Rust for Linux supports, depending on the Linux version. Two sets are provided for each supported kernel release: the "latest LLVM" and "matching LLVM" (please see the link for more information).

They are also now linked in the Rust for Linux webpage (left side, in "Links" -> "Toolchains"), as well as in the kernel documentation Quick Start guide soon (patch).
