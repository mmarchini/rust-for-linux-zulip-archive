Hi All, we plan to change a few "Streams" from "public" (anyone can access but needs an account) to "web-public" (anyone can access without an account, including a search engine I assume), (details). The reasons are:

  • Since these streams are already public and anyone can register an account in this zulip, so it makes little difference if they are web-public.
  • Being web-public makes it possible to reference the discussion in this zulip in git commit logs, emails, etc.

The streams that are going to be "web-public" are (along with "Networking", which is already web-public):

  • Announcements
  • Filesystems
  • General
  • Help
  • Library
  • Meetings
  • Modules

And of course, any newly created stream will be "web-public" as well

The changes are going to happen at Oct 1st, 2023, and if you have any question or concern, please let us know (feel free to send me a direct message in case of privacy issues). Thanks!
