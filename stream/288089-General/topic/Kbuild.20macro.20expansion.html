bjorn3 (Jan 18 2022 at 20:21):

Rustc has the unstable -Zunpretty=expanded flag for this. You will probably want to run the output through rustfmt though (the default formatting is based on a very old style and has a lot of rightward drift. expect lines >100 chars for anything using format_args!()) It can also get pretty long if you use many modules as they all get inlined.
