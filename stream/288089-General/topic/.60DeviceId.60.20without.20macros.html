Gary Guo (Sep 22 2024 at 13:58):

Two key tricks here:

  1. Use MaybeUninit and transmute_copy to essentially implement array unzip using const fn. This avoids macros for first/second element.
  2. Use transmute_copy + raw pointer ops to avoid to_rawid function, so it neither needs macro nor const trait impl
