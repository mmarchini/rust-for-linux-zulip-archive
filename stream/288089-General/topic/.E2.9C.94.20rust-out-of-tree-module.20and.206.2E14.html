Miguel Ojeda (Mar 27 2025 at 11:41):

Is there an older kernel I should have used? Would you like a PR if I work out how to update the example repo to work with Linux 6.14?

That repository tracks the mainline kernel, not particular releases. In other words, it is not that the repository needs an update, but rather that it is already updated to use the new way.

Please see the commit log there, e.g. the latest one is the one that changed that to match mainline:

rust_out_of_tree: use the new authors key

Soft-required since 38559da6afb2 ("rust: module: introduce authors key") [1] -- it may be actually required in a kernel cycle or two.

Signed-off-by: Miguel Ojeda <ojeda@kernel.org>
