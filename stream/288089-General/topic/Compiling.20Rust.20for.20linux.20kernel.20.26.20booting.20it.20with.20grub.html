Valentin Obst (Feb 24 2024 at 22:25):

If you just want to quickly boot a kernel in a VM for some basic testing I recommend using virtme-ng. It uses copy-on-write versions of your host file system mounts in the VM so you don't even need to build a rootfs or initrd. In particular, this means that any modules that you build are easily accessible for loading.

I want to enable rust in the kernel of my VM. Fortunatly I managed to solve it by using the .config of 6.3.0-generic and changing "MODVERSIONS" to n in my .config file and manually adding CONFIG_RUST=y, though I don't know if this is the right way to do it.
