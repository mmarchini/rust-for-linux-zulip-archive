Boqun Feng (Mar 14 2023 at 14:33):

A tricky part for this is class_create macro allocates a static location for lock_class_key, so to behave as the same, you will need introduce a Rust macro, for example

macro_rules! class_create {
  (owner:$expr, name:$expr) => {{
    static key: LockClassKey = ...;
    // create local binding to avoid evaluate unknown expression in `unsafe`
    let owner = $owner;
    let name = $name;
    unsafe {bindings::__class_create(owner, name.as_char_ptr(), KEY.get())}
  }}
}

Boqun Feng (Mar 14 2023 at 15:16):

I think LockClassKey::get() is both public and safe, so maybe you can define a "constructor" for Class:

impl Class {
  pub fn new(name: &'static CStr, key: &'static LockClassKey) -> Self {
    let mut owner = Class { .. };

    unsafe {bindings::__class_create(owner.0, name.as_char_ptr(), key.get())}
    owner
  }
}
