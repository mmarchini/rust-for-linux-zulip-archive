Topic: How do I compile a Linux kernel for the aarch64 platform

Peng He (Mar 31 2024 at 17:12):

want to build a bin file to raspberrypi4b or rk chip, and develop some driver. In rust-dev branch, I saw that the arch-support.rst document says that aarch64 is supported, and how can I compile a Linux kernel for the aarch64 platform.

Fabien Parent (Mar 31 2024 at 17:40):

You can look at the Quick start: https://docs.kernel.org/rust/quick-start.html

The only difference with what's on the "Quick Start" is that you need to change the ARCH since it defaults to x86.

You can use these commands:

$ make LLVM=1 ARCH=arm64 defconfig rust.config
$ make LLVM=1 ARCH=arm64 rustavailable
Rust is available!
$ make LLVM=1 ARCH=arm64
