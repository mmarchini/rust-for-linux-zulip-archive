Wedson Almeida Filho (Old Google account) (Mar 31 2022 at 15:35):

I think it could help if it covered the case where the return value has a local binding in the function. This example is taken from the rendered RFC:

// This function may or may not copy the array when it returns.
fn bad() -> [i32; 1_000_000] {
    let mut arr = [0; 1_000_000];
    for i in 0..1_000_000 {
        arr[i] = i;
    }
    arr
}

It explicitly says that it may not work, but this is the sort of thing we'd need to help with pinning because we need to pass a pointer to mutex_init, which is written in C.
