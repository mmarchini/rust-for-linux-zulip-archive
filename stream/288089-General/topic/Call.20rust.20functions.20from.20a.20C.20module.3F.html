Topic: Call rust functions from a C module?

Jayaura (Sep 13 2024 at 15:50):

Is it possible today to have the main driver in C but call some functions that are written in Rust? Examples I've seen have the full driver in Rust. If this is possible, I could experiment with converting isolated parts in the driver over to Rust

Alice Ryhl (Sep 13 2024 at 15:53):

Yes, but you have to be careful to make the boundary as small as possible. If your Rust code is littered with calls into C, then that's not great.

Alice Ryhl (Sep 13 2024 at 15:54):

The QR code panic module that was recently merged is an example. The logic to generate the QR code is in Rust, but the rest is in C. The interface between the languages is very small, as it literally just passes a buffer from C to Rust and back.

Alice Ryhl (Sep 13 2024 at 15:55):

The Rust Binder driver is another example. The binderfs component is written in C, but calls into Rust code.
