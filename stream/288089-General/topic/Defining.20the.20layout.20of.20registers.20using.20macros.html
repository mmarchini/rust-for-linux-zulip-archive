Benno Lossin (Mar 28 2025 at 01:04):

can you post the code that you tried? It should be pretty simple, create a new function in rust/macros/lib.rs with the name of your macro, put #[proc_macro] on top of it and then delegate to some file in which you define your proc macro like the other files in rust/macros.

Appending this to rust/macros/lib.rs works for me:

/// foo
#[proc_macro]
pub fn foo(input: TokenStream) -> TokenStream {
    quote! {}
}
