Thoughts about changing cursors so that they point between two elements, instead of pointing at an element?

The current cursor API is pretty inconvenient to use. You almost always have to deal with an optional cursor everywhere. Taking an idea from the stdlib linked list, change the cursor so that it points between two elements instead of pointing at an element. This way, all of the methods become &mut self methods instead of the much less convenient signatureSelf -> Option<Self>.

https://github.com/Darksonn/linux/commits/b4/cursor-between/
