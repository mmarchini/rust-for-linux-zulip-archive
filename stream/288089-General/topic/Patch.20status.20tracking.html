git log --grep is actually quite fast. Lets say you have this patch and you want to know if it is applied:

time git log --grep "rust: macros: replace" --oneline rust/rust-next
52b7bb46ae79 rust: macros: replace Self with the concrete type in #[pin_data]
git log --grep "rust: macros: replace" --oneline rust/rust-next  7.84s user 0.11s system 99% cpu 7.956 total

time git log --grep "rust: macros: replace" --oneline upstream/master
git log --grep "rust: macros: replace" --oneline upstream/master  7.82s user 0.15s system 99% cpu 8.014 total

It took me 14 seconds to figure out that it is staged in rust-next.

Lore also has a really good search function with an expressive query language. It is quite easy to find email threads for specific changes with that.
