Trevor Gross (Aug 16 2023 at 23:33):

By "introduce" do you mean why are there hostprogs in the kernel at all?

It doesn't matter if they're written in C or Rust. Hostprogs are compiled programs that do something related to the kernel but just aren't part of the kernel image. Things like:

  • Build configuration
  • Generate source files
  • Test setup / running

They are compiled by kbuild. Before RFL, you would write hostprogs in C but now, you can also write them in Rust

Trevor Gross (Aug 16 2023 at 23:35):

The code is no different than any other Rust binary that you might write. The source is just in linux and they are built by the kernel build system
