Say we have a

extern "C" pub f(..) {
    ...
}

Could callsites to it be inlined (with LTO)? What if f has asm in it? @bjorn3 @Gary Guo

bjorn3 (Aug 22 2022 at 21:17):

With cross language LTO it can be inlined between C and Rust. Note however that LTO is not currently implemented for Rust code I believe. Passing -Clinker-plugin-lto to all rustc invocations may work if you are building with LLVM. I don't think anyone has tried that before though.
